## **Setup & Installation**

1. Clone the repository:  
   ```bash
   git clone https://github.com/Hesara2003/E-commerce.git
   cd E-commerce
   ```

2. Install dependencies:  
   ```bash
   npm install
   ```

3. Start the development server:  
   ```bash
   npm start
   ```

4. Run the app:  
   - On an iOS/Android emulator or a physical device using Expo Go.
